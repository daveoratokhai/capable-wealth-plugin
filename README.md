# Capable Wealth — Cowork Plugin Marketplace

Private marketplace for the Capable Wealth content harness (seven skills). The skills are self-contained; the only external capability used is web search, by `/research-scan`, which must be enabled in the Cowork project. Slack is optional.

**Install in Cowork:** Personal plugins → Add marketplace → this repo's URL → install `capable-wealth`.

The plugin's skills read the brand docs (`brand/`, `rules/`, etc.) from the **linked Cowork project folder** at runtime, so this repo intentionally contains only the skills + marketplace catalog.
